# SADC 电池电压检测示例

## 概述

本示例演示如何使用 B6x 系列 MCU 内置的 SADC（逐次逼近型 ADC）
测量电池电压。支持三种测量模式：

- **VDD 模式**（默认）：通过内部 VDD12 通道直接测量 VDD
  供电电压，无需外部引脚
- **锂电池模式**：通过外部 ADC IO 引脚测量锂电池电压，
  使用 2.4V 内部基准 + 等比分压电阻
- **DCDC 模式**：通过外部 ADC IO 引脚测量 DCDC 输出电压，
  使用 1.2V 内部基准 + 2:1 分压电阻

程序持续读取 ADC 采样值，经过去极值滤波算法后计算出实际电压
（单位 mV），并通过串口/RTT 输出。

## 配置说明

编辑 `src/cfg.h` 进行配置：

| 宏定义              | 默认值     | 说明                             |
| ------------------- | ---------- | -------------------------------- |
| `DBG_MODE`          | `1`(UART)  | 调试输出：0=关闭，1=UART，2=RTT  |
| `LI_BATT_MODE`      | `0`        | 启用锂电池测量模式               |
| `DCDC_MODE`         | `0`        | 启用 DCDC 测量模式               |

## 硬件连接

### VDD 模式（默认）

无需外部连接。SADC 通过内部通道直接采样 VDD12，反推 VDD33 电压。

```text
┌──────────────────┐
│                  │
│   B6x MCU        │
│                  │
│   内部 VDD12 ──→ SADC_CH_VDD12
│                  │
└──────────────────┘
```

### 锂电池模式 (LI_BATT_MODE = 1)

使用 R1 = R2 等比分压电路，将电池电压衰减一半后送入 ADC 引脚。

```text
                    R1 (1MΩ)
  V_BAT ──────────╱╱╱╱╱╱╱╱╱╱╱╱╱╱───────┬─────── PA13 (AIN2/V_adc)
                                       │
                                    R2 (1MΩ)
                                       │
                                      GND

  分压比: V_adc = V_bat / 2
  量程:   V_bat_max = 2.4V × 2 = 4.8V
```

### DCDC 模式 (DCDC_MODE = 1)

使用 R1:R2 = 2:1 的分压电路，将 DCDC 输出电压衰减为 1/3
后送入 ADC 引脚。

```text
                    R1 (2MΩ)
  V_DCDC ─────────╱╱╱╱╱╱╱╱╱╱╱╱╱╱───────┬─────── PA13 (AIN2/V_adc)
                                       │
                                    R2 (1MΩ)
                                       │
                                      GND

  分压比: V_adc = V_dcdc / 3
  量程:   V_dcdc_max = 1.2V × 3 = 3.6V
```

## 测量原理

### SADC 架构

B6x 内置 10 位逐次逼近型 ADC (SAR ADC)，分辨率为 1024 级 (0~1023)。
基本转换关系：

```text
ADC_code = V_input × 1024 / V_reference
```

SADC 支持多种基准电压源，通过 `sadc_init(ana_ctrl)` 选择：

| 基准源             | 宏定义              | 典型值  | 适用场景                |
| ------------------ | ------------------- | ------- | ----------------------- |
| VDD 供电电压       | `SADC_ANA_VREF_VDD` | ~3.3V   | 测量内部通道，反推 VDD  |
| 内部带隙 1.2V      | `SADC_ANA_VREF_1V2` | ~1.2V   | 外部低电压测量          |
| 内部带隙 2.4V      | `SADC_ANA_VREF_2V4` | ~2.4V   | 外部较高电压测量        |

> 基准电压来自芯片内部带隙基准 (VBG) 或 VDD 引脚，无需外部基准电路。

### VDD12 校准基准

B6x 出厂时对内部 1.2V 稳压器 (VDD12) 进行精确校准，校准值存储在
FT (Factory Trim) 区域。`get_trim_vdd12_voltage()` 返回该校准值（单位 mV），
作为所有电压计算的参考锚点。

**核心要点**：不论使用哪种基准电压源，VDD12 的校准值始终是推算实际电压的
已知量。内部带隙基准与 VDD12 有确定的倍数关系（1.2V = VDD12 × 1，
2.4V = VDD12 × 2），因此可从 VDD12 校准值推导出所有基准电压的精确值。

### 三种测量模式的工作原理

#### 1. VDD 模式 — 测量供电电压

```text
                    ┌──────────────────────────┐
                    │                          │
  VDD33 (未知)  ────┤   B6x MCU                │
                    │                          │
                    │   内部 LDO               │
                    │   VDD33 → VDD12 (已知)   │
                    │            │             │
                    │            ▼             │
                    │   SADC_CH_VDD12 ──→ ADC  │
                    │            ▲             │
                    │            │             │
                    │   基准: VDD33 (未知)     │
                    │                          │
                    └──────────────────────────┘

  ADC = VDD12 × 1024 / VDD33
        ↓ 反推
  VDD33 = VDD12 × 1024 / ADC
```

- **基准源**：VDD33 供电电压本身
- **采样通道**：内部 `SADC_CH_VDD12`（无需外部引脚）
- **原理**：利用 "已知量测未知量" 的反测法 — VDD12 是已知的稳定电压，
  以未知的 VDD33 为基准测量 VDD12，ADC 值反比于 VDD33
- **限制**：只能测量 VDD33 供电电压，无法测量独立电池电压

#### 2. 锂电池模式 — 测量外部锂电池

```text
                    ┌──────────────────────────┐
                    │                          │
  V_BAT (未知) ───→ │   外部分压电路           │
                    │   R1=R2 → V_adc = V_BAT/2│──→ PA13 (AIN2)
                    │                          │     │
                    │   B6x MCU                │     ▼
                    │                          │   SADC
                    │   基准: VBG = VDD12 × 2  │     │
                    │        (≈ 2.4V)          │     ▼
                    │                          │   ADC_code
                    └──────────────────────────┘

  ADC_code = V_adc × 1024 / VREF
           = (V_BAT / 2) × 1024 / (VDD12 × 2)
        ↓ 整理
  V_BAT = VDD12 × 2 × ADC_code × 2 / 1024
```

- **基准源**：内部带隙 2.4V (= VDD12 × 2)
- **采样通道**：外部 `SADC_CH_AIN2` (PA13)
- **分压电路**：R1 = R2 = 1MΩ，分压比 1:1，V_adc = V_BAT / 2
- **量程**：V_BAT_max = 2.4V × 2 = 4.8V，覆盖单节锂电池 3.0V~4.2V 全范围

#### 3. DCDC 模式 — 测量 DCDC 输出电压

```text
                    ┌──────────────────────────┐
                    │                          │
  V_DCDC (未知) ──→ │   外部分压电路           │
                    │   R1:R2=2:1 → V_adc=V/3  │──→ PA13 (AIN2)
                    │                          │     │
                    │   B6x MCU                │     ▼
                    │                          │   SADC
                    │   基准: VBG = VDD12      │     │
                    │        (≈ 1.2V)          │     ▼
                    │                          │   ADC_code
                    └──────────────────────────┘

  ADC_code = V_adc × 1024 / VREF
           = (V_DCDC / 3) × 1024 / VDD12
        ↓ 整理
  V_DCDC = VDD12 × ADC_code × 3 / 1024
```

- **基准源**：内部带隙 1.2V (= VDD12)
- **采样通道**：外部 `SADC_CH_AIN2` (PA13)
- **分压电路**：R1:R2 = 2:1，V_adc = V_DCDC / 3
- **量程**：V_DCDC_max = 1.2V × 3 = 3.6V

### 基准电压与分压比的匹配

SADC 输入电压不得超过所选基准电压（VREF）。设计时需根据目标电压范围
选择合适的分压比和基准源：

```text
  目标电压      分压后         基准电压        是否满足 V_adc ≤ VREF?
  ─────────────────────────────────────────────────────────────
  锂电 4.2V  →  ÷2 = 2.1V  →  VDD12×2 ≈ 2.4V   ✓ (留 0.3V 余量)
  DCDC 3.3V  →  ÷3 = 1.1V  →  VDD12   ≈ 1.2V   ✓ (留 0.1V 余量)
  VDD        →  内部通道   →  VDD              ✓ (自身作基准)
```

### 滤波算法

电压测量采用去极值平均滤波，消除偶发噪声干扰：

1. 连续采样 N 次 (N = SADC_READ_CNT = 6)
2. 剔除最大值和最小值（排除偶发跳变）
3. 对剩余 N-2 个值取平均
4. 将平均值代入对应模式的电压公式

```text
  采样序列: [a, b, c, d, e, f]
            ↓ 排序找极值
  剔除 min(a..f) 和 max(a..f)
            ↓ 剩余 4 个值
  ADC_avg = (sum - max - min) / (N - 2)
```

## 算法说明

电压测量采用去极值平均滤波：

1. 连续采样 6 次 ADC 值
2. 去掉最大值和最小值
3. 对剩余 4 个值取平均
4. 根据当前模式套用电压计算公式

### 电压计算公式

| 模式   | 采样通道  | 基准 | 计算公式              |
| ------ | --------- | ---- | ----------------------|
| VDD    | CH_VDD12  | VDD  | `(VDD12×1024)/ADC`    |
| 锂电池 | CH_AIN2   | 2.4V | `(VDD12×2×ADC×2)/1024`|
| DCDC   | CH_AIN2   | 1.2V | `(VDD12×ADC×3)/1024`  |

### 公式推导

以锂电池模式为例：

```text
V_adc = VREF × ADC / 1024           (ADC 基本转换)
      = (VDD12 × 2) × ADC / 1024    (2.4V 基准 = VDD12 × 2)

V_bat = V_adc × (R1+R2) / R2        (分压还原)
      = V_adc × 2                    (R1 = R2)
      = VDD12 × 2 × ADC × 2 / 1024
```
