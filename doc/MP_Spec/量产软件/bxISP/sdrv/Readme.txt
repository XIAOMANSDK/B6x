/******************************************************************************
 * sdrv文件说明(*.bin)
 ******************************************************************************/
1.  burner[T]_ver[N]_hal[W].bin -- 适配器(烧录器)固件
    其中，[T]=B6x为芯片型号，[N]为固件版本号，[W]为硬件版本号

2.  chipset[T]_ver[N]_[Info].bin -- 芯片加载驱动
    其中，[T]=B6x为芯片型号，[N]为驱动版本号，[Info]为附加信息

3.  empty0xFF.bin -- Empty Flash Page(256B filled with 0xFF)
